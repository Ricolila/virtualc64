\doxysubsection{Detailed Description}
Implements the emulator\textquotesingle{}s state model. 

This class is one of the base classes of the Emulator class and provides the basic functionality to manage the emulator state. It provides functions to launch the emulator thread, to query it\textquotesingle{}s current state, and to switch to another state.
